# password_manager

## 概要
シェルスクリプトで作ったパスワードマネージャー

## 使い方
./password_manager.shを実行する

## 機能
- Add Password: パスワードの追加
- Get Password: パスワードの取得
- Exit: 終了

## 実行環境
- Git Bash: WindowsでGit・Linuxコマンドを使う
- GCP(Google Cloud Platform): クラウド上でのLinux環境
- Ubuntu 22.04: GCP上でのOS

## 暗号化
- GPG(GnuPG): ファイルの暗号化・復号化
